# Netflix-Data-Analytics-Visualization-Dashboard
### Project Overview
This project performs a deep-dive Exploratory Data Analysis (EDA) on the Netflix Titles dataset. The goal is to uncover trends in content additions, regional production strengths, and the evolution of the platform's library from 1940 to 2021.

### Key Features
Automated Data Cleaning: Future-proofed for Pandas 3.0 (handles chained assignments and null value management).

Feature Engineering: Extracted numeric durations and converted date strings into datetime objects for time-series analysis.

Executive Dashboard: A comprehensive 4-panel visual report saved as high-resolution PNG/PDF.

Content Recommender: A rule-based engine to suggest movies/shows based on Genre and Rating similarity.

###  Visualizations Included
Content Distribution: Pie chart showing the Movie vs. TV Show ratio.

Global Production: Bar chart identifying the Top 10 content-producing countries.

Rating Heatmap: Relationship between target audiences and countries.

Growth Trends: Line chart depicting content additions per year.

Genre Word Cloud: Visual representation of the most frequent genres.
